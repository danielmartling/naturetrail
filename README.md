# naturetrail

Webbsidor, ljudfiler och qr-koder för Vässarös naturspår "Naturnäran". Navigationen är bortplockad för att göra sidan "enkel", men det finns en hemlig [https://vassaro.github.io/naturetrail/nav/](innehållsförteckning).

## För att bidra:

1. I en python-venv, installera mkdocs, mkdocs-material, mkdocs-glightbox, mkdocs-audio och mkdocs-video.
2. Gör ändringar i `/src/docs/` och kör sen `mkdocs build` i `/src/`.

- Kommandot `mkdocs serve` ger en live preview av sidan.
- Ljudklipp kan läggas i `/src/docs/audio/`.
